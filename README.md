## Media Collection Manager Description

The purpose of this project is to develop a functional at-home media management system that allows users to fully control and organize their personal music collections. This program enables users to manage physical music media by keeping track of artists, albums, and songs within their library. It includes search functionality to quickly locate specific items, as well as features to save and load libraries through file reading and writing. This functionality allows users to maintain and access multiple collections efficiently.

## How to Run
* Go to Releases section and download runnable `.jar` file
* `File` Menu contains features such as `Save` `Save As` `Open` and an `Auto Save` toggle feature.
* Search Functionality is text entry field on top. If Artist is searched > returns Albums. If Album is searched > return Album. If Song is searched > return Song title and Duration.
* Click left `+` to add Artist to Artist Panel. - Dialog box appears.
* While in Album view click right `+` to add an Album to Album Panel. - Dialog box appears.
* While in Song view click `+` one more time to add individual Songs to an Album.
* The `Save` and `Load` buttons provide easier access to our `Save As` and `Open` function.
* All saving and opening function are assisted by a file directory.
